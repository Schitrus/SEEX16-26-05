# Mäta avstånd till döende AGB-stjärnor
Detta kandidatarbete ämnar till att beräkna avstånd till döende stjärnor av typen *asymptotic giant branch* (AGB).

De stjärnor som kommer att observeras är R Doradus, R Leonis, W Hydrae och data fås från ALMA-teleskopen.

Beräkningen sker i tre steg:
1. Anpassning av intensitetbilder från observationer för att få fram stjärnornas mittpunkt.
2. Plottandet av stjärnornas rörelse i IRCF-systemet där parallaxen samt egenrörelsen kommer att framträda.
3. Beräkning av avståndet från parallaxen.

Analysen och beräkning sker med Python (3.14.2) och följande Python-bibliotek används
- Numpy 2.4.2
- Scipy 1.17.0
- Matplotlib 3.10
- Pandas 3.0.0
- Astropy 7.2.0
